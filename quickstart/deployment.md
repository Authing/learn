# 部署模型和价格

Authing 提供两种部署方式：

| 部署方式 | 描述 |
| :--- | :--- |
| 公有云 | 运行在多租户的 Authing Cloud 中 |
| 私有云 | 运行在客户自己的基础设施中或其他云环境 |
| 专有云 | 运行在 Authing 开辟的单租户公有云环境中 |

如果你有私有部署的需求请联系：+86 17602502507。

## 公有云价格说明

Authing 的公有云按认证次数计费。

| 认证次数 | 每次认证价格 |
| :--- | :--- |
| 第一个 8000 | 免费 |
| 下一个 20000 | 0.0195 元人民币 |
| 下一个 20000 以上 | 0.018 元人民币 |
