# 邮箱 - 密码认证

邮箱 - 密码认证指使用邮箱 + 密码的形式完成认证：

![&#x4E00;&#x4E2A;&#x4F7F;&#x7528;&#x90AE;&#x7BB1;&#x8BA4;&#x8BC1;&#x7684;&#x793A;&#x4F8B;](../../.gitbook/assets/image%20%28460%29.png)

用户使用邮箱 - 密码进行认证，在注册后将引发以下事件：

1. 用户将收到一封欢迎邮件
2. 用户还将收到一封确认邮件，用户点击确认邮件中的链接后可完成邮箱验证
