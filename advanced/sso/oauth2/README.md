# 使用 OAuth 2.0

通过 OAuth 2.0 可以将 Authing 用户池作为身份服务提供商，使第三方应用可以读取你的用户数据，流程与 Github 基本一致。我们提供了 OAuth 认证流程的四种模式，分别是 `implicit 模式`、`authorization_code 模式`、`password 模式`和 `client_credentials 模式`。
